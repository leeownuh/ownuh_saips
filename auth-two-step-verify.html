TOTP Authentication

Time-based one-time passwords via Google Authenticator or Authy. 30-second window with one-step tolerance (RFC 6238 compliant).

FIDO2 / WebAuthn

Hardware security keys (YubiKey, etc.) are required for all CEO and Superadmin accounts — the strongest available second factor.

Backup Recovery Codes

Ten single-use backup codes are generated at MFA enrollment, stored as bcrypt hashes. Keep them in a safe location.

Two-Factor Authentication

Enter the 6-digit code from your authenticator app, or the code sent to s••••••@acme.com.
